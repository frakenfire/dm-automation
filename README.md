# 🚀 P.CALM Instagram Seeding Dashboard

인스타그램 인플루언서 시딩을 위한 프리미엄 웹 관제 대시보드입니다. 터미널의 복잡함 없이 웹 UI를 통해 실시간으로 발송 상태를 모니터링하고 제어할 수 있습니다.

## 🛠 실행 방법 (내일 아침 바로 시작하기)

1. **터미널 열기**: 프로젝트 루트 폴더(`디엠 자동화`)에서 터미널을 엽니다.
2. **서버 가동**: 아래 명령어를 입력합니다.
   ```bash
   npm run dashboard
   ```
3. **대시보드 접속**: 브라우저 주소창에 아래 주소를 입력합니다.
   ```
   http://localhost:3000
   ```

## 🎯 사용 가이드

- **타겟 핸들**: @를 제외한 인스타그램 아이디를 입력하세요.
- **발송 메시지**: 인플루언서에게 보낼 정성스러운 메시지를 작성하세요.
- **백그라운드 실행**: 비권장 사항입니다. 브라우저 창이 직접 뜨는 것을 확인하며 발송하는 것이 가장 안전합니다.
- **실시간 로그**: 우측의 넓은 로그 창을 통해 로봇이 현재 어느 단계(검색, 선택, 작성 등)에 있는지 실시간으로 확인할 수 있습니다.

## 📅 내일의 과제 (Upcoming Features)

오늘 구축된 단일 발송 인프라를 바탕으로 내일은 아래 기능을 확장할 예정입니다:

1. **다중 발송 자동화**: 구글 시트나 CSV 리스트를 업로드하여 로봇이 수십 명에게 순차적으로 발송하는 기능.
2. **진행률 최적화**: 다중 발송 시 전체 작업의 완성도를 보여주는 게이지 고도화.
3. **결과 리포트**: 발송 성공/실패 여부를 자동으로 기록하여 최종 리포트 생성.
